	Keyframes
		Click stop watch to animate a property; such as rotation, opacity, position etc.
			Only one frame is needed, and the rest get created automatically.
		
		Keyframes can be copied, moved and deleted, multiple or single.
	
	Making Animation Better:
		Motion Blur
			Enable motion blur (twice)
		
		Ramping Up and Down:
			Easy Ease F9
			or:
				Right Click the keyframe; and chose Keyframe Assistant; then Easy Ease
				Or you can input the velocity using right-click (Keyframe Velocity) - Default value is 33.333
				Choose higher values like 75.
				
			
			Notice how the  keyframe changes its symboel from dimond to I.
		
		Graph Editor:
			First select the keyframes you want to edit in the graph editor.
				Make sure you're editing the speed [Right-Click	>>>	Edit Speed Graph].
